# installer

CLI tool to install some of my favourite Linux apps based on the distro (debian,arch or fedora).

#

## Prerequisites/ Dependencies

1.  Any GNU/Linux distribution based on Debian/Ubuntu, Arch, Red Hat/Fedora.
2.  `git` , `make`
